This is an MLOps submission project for kidney stone detection using urine analysis. It consists of two phases:

1. **Model Building** (`Membangun_model/`) — trains a RandomForestClassifier and tracks experiments via MLflow/DagsHub
2. **Monitoring & Logging** (`Monitor_dan_logging/`) — serves the model and collects metrics via Prometheus + Grafana

- GitHub (experiments): https://github.com/syarif-nur/Eksperimen_SML_Syarif-Nur
- GitHub (CI workflow): https://github.com/syarif-nur/submission_msml
- DagsHub MLflow tracking: https://dagshub.com/syarif-nur/urine-analysis.mlflow
